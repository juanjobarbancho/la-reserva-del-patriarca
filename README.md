# La Reserva del Patriarca

Web de promoción inmobiliaria para la cooperativa de viviendas "La Reserva del Patriarca" en Córdoba.

## Características

- 37 viviendas de diseño moderno
- 2-4 dormitorios
- Piscina comunitaria
- Zonas verdes y jardines privativos
- Todas las viviendas incluyen trastero y plaza de garaje

## Tecnologías

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Supabase

## Desarrollo

```bash
npm install
npm run dev
```

## Deploy

El proyecto está configurado para deploy automático en Vercel.
